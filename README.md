# GraveFinder SA

## 🇿🇦 Helping Families Locate Graves Across South Africa

GraveFinder SA is a web and mobile platform that helps families search for and locate the graves of their loved ones.

### Features
- 🔍 Search for graves
- 📍 GPS directions to graves
- 🪦 Grave information
- 🖼️ Grave photos
- 🏛️ Cemetery directory
- 👤 Secure admin login

### First Launch
The first version of GraveFinder SA will focus on cemeteries in Soweto, Gauteng, South Africa.
